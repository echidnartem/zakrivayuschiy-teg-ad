# Закрывающий тег

Страница-воспоминание о пройденном курсе с анимированными лайками. Учебный проект по CSS-анимациям и модальным окнам.

---

## 📋 О проекте

Проект представляет собой галерею карточек с воспоминаниями о спринтах курса. Каждая карточка содержит изображение, текст и кнопку лайка с анимированным сердцем. Внизу страницы есть кнопка "Сохранить на память", открывающая модальное окно.

Вся логика работы лайков была предоставлена в готовом виде. Моя задача заключалась в вёрстке адаптивного интерфейса по макету Figma, настройке CSS-анимаций и позиционировании элементов.

**Особенность:** 
- Сердце при лайке анимированно увеличивается в размерах и "взрывается" искрами.
- Для изображений применены различные CSS-фильтры (`grayscale`, `saturate`, `hue-rotate`, `contrast`, `brightness`, `sepia`).
- Реализовано модальное окно с использованием тега `<dialog>`.
- Кнопки имеют анимированный фон при наведении.
